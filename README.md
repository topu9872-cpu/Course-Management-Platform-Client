# 🎓 CourseHub Client

A modern, full-stack **Course Management Platform** built with **Next.js 16**, **React 19**, **TypeScript**, and **Tailwind CSS**. CourseHub enables students to discover courses, instructors to manage their content, and administrators to oversee the entire platform through a secure, role-based dashboard.

# 📦 Installation

Clone the repository

```bash
git clone https://github.com/your-username/coursehub-client.git
```

Navigate to the project

```bash
cd coursehub-client
```

Install dependencies

```bash
npm install
```

Create a `.env.local` file.

```env
NEXT_PUBLIC_SERVER_URL=https://programming-hero-a-11-course-manage.vercel.app

NEXT_PUBLIC_APP_URL=https://course-management-platform-client-4i05sl3f6.vercel.app
BETTER_AUTH_SECRET=your_secret

GOOGLE_CLIENT_ID=your_google_client_id

GITHUB_CLIENT_ID=your_github_client_id

FACEBOOK_CLIENT_ID=your_facebook_client_id

NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=your_publishable_key
```

Run the development server

```bash
npm run dev
```

Open

```text
http://localhost:3000
```
